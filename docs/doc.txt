Утилиты написаны для работы в следующей сетевой конфигурации:
Единый логин и пароль для всех устройств обрабатываемой группы.
AAA устройств настроено таким образом, что после авторизации под указанным логином,
устройства автоматически переходят в режим "enable" без ввода дополнительного пароля.

Запуск всех утилит с ключом -h выводит описание её назначения и перечень всех обязательных и опциональных параметров.

Файлы с перечнем устройств составляются в формате: одна строчка - один ip; всё, что не ip - комментарий.
Файлы со словарями сохраняются и читаются в формате yaml.
Для обработки результатов вывода команд "sh ..." и "display ..." используюся шаблоны textfsm.
Для создания команд конфигурации используюся шаблоны jinja2 и текстовые файлы.

Импортируются следующие пакеты:
- os
- ipaddress
- getpass
- yaml
- logging
- re
- graphviz
- tabulate
- pprint
- netmiko
- textfsm
- jinja2
- argparse
- subprocess
- concurrent.futures
- itertools
- csv
- dns
